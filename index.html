DisplayWindowManager

一键操控多显示器窗口 —— 断开副屏后窗口不再"卡"在看不见的地方，轻量、免费、开源

立即下载 (加载中...) Windows 10/11 · 双击运行 · 免费开源 GitHub 仓库
核心功能

一键移动窗口

点击或按 Ctrl+Shift+M，批量将所有窗口移动到目标显示器，DPI 自动适配

双向移动

支持副屏→主屏和主屏→副屏两种方向，随时切换

显示器屏蔽

Ctrl+Shift+S 开关，阻止新窗口进入指定显示器，自动移走

多语言界面

中/英文一键切换，语言偏好自动保存

系统托盘

最小化到托盘，右键菜单快捷操作，全局热键随时触发

零依赖单文件

纯 Python + Win32 API，打包为 ~6.6MB 单文件 EXE，无需安装

快速开始

方式一：下载 EXE（推荐）

下载 DisplayWindowManager.exe
加载中...

— 或从源码运行 —

git clone https://github.com/Fake-msn/DisplayWindowManager.git
cd DisplayWindowManager
python display_window_manager.py

方式三：自行打包

pip install pyinstaller
pyinstaller --onefile --windowed --name "DisplayWindowManager" display_window_manager.py

已知限制

部分系统级窗口（任务管理器、系统设置、UAC 弹窗等）受 Windows 完整性级别隔离（UIPI）安全机制保护，无法被第三方软件移动。这是操作系统的安全设计，所有同类工具均受限。绝大多数日常应用的窗口均可正常移动。
