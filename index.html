// 核心功能

精准控制每一个显示器

专为多显示器用户设计的轻量工具，解决远程桌面断开、副屏拔插后的窗口管理痛点。

一键移动窗口

点击按钮或 Ctrl+Shift+M，所有窗口批量移至目标显示器，DPI 智能缩放自动适配

双向移动

副屏→主屏 / 主屏→副屏，设置中随时切换方向，灵活适配不同场景

显示器屏蔽

Ctrl+Shift+S 开关，阻止新窗口进入指定显示器，自动移走已有窗口

多语言界面

中/英文一键切换，语言偏好自动保存，架构可扩展更多语言

系统托盘常驻

最小化到托盘，右键菜单快捷操作，全局热键随时触发，安静不打扰

零依赖单文件

纯 Python + Win32 API，打包为 ~6.6MB 单文件 EXE，无需安装，即开即用

// 已知限制

系统级窗口无法移动

部分系统级窗口（任务管理器、系统设置、UAC 弹窗等）受 Windows 完整性级别隔离（UIPI）安全机制保护，无法被第三方软件移动。这是操作系统的安全设计，所有同类工具均受限。绝大多数日常应用的窗口均可正常移动。
